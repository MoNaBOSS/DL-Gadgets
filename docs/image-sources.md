# Product image sources

The browsing service was unavailable during this redesign, so the initial catalog imagery is original AI-generated commercial product mockup work. It is stored locally, is not hotlinked, includes no manufacturer logo, and is deliberately used as category-level imagery for models without verified official product photography.

| Used by | Filename | Source | Notes |
| --- | --- | --- | --- |
| Smartphones and clearance/used devices | `public/products/smartphone-studio.webp` | Original DL Gadgets catalog visual generated locally | Neutral smartphone presentation, not an exact Apple, Samsung, Motorola, or Huawei model. |
| Hero and broad category imagery | `public/products/electronics-collection.webp` | Original DL Gadgets catalog visual generated locally | Tablet, gaming handheld, security camera, and earbuds collection. |
| Category fallbacks | `public/placeholders/*.webp` | Derived from original DL Gadgets collection visuals | High-quality local category-level fallback images. |

## Replacement process

Replace category-level fallbacks with verified official manufacturer or licensed product photography as individual product assets become available. Record each future product slug, filename, source URL, and licence/usage note in this document before publishing.
